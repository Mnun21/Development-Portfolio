About Me

Hello! My name is Moises. I am currently working as a tech consultant but have an ambition to become a Full Stack Web Developer. I developed this interest when learning basic programming skills using coding languages such as Java and C. When I found out about the Thinkful Engineering Immersion program I knew I could not wait on taking advantage of this opportunity.

At my job as a tech consultant I learned that my passion is to help people and that open team collaboration is key not only to an organization's success but also to the individual. My goal is not only to get a 9 to 5 job but to find a workplace/community that will allow me to grow and develop more impactful skills as a software engineer. One individual project that I hope to complete is an educational and responsive web app that helps break down complex physics topics so that anyone with a small interest can learn something new.
